# 쿠키(Cookie)

**장점**

- 서버의 저장공간 절약

**단점**

- 보안 취약 : 요청 시 쿠키 값을 그대로 보냄
- 작은 허용 용량 : 사이트 당 20개, 모두 합쳐 300개가 최대, 각 쿠키는 4KByte를 넘을 수 없음
- 웹브라우저마다 지원 형태가 다름
- 웹브라우저를 변경할 경우 다른 웹브라우저에서 저장한 쿠기값을 사용할 수 없음
- 사용자가 보안상의 문제로 거부할 경우 사용 불가능
- 네트워크 부하 : 쿠키의 크기가 클 경우 네트워크 부하가 커짐
- 한번에 하나의 정보만 저장할 수 있음
